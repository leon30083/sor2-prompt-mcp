现在我的制作流程如下:
1. 输入剧本原稿
2. 对剧本原稿进行剧本格式化处理，添加每个片段的标题。
3. 根据格式化后的剧本生成每个片段的sora2视频指令，指令格式如下:

剧本结构如下：
每个剧本章节 都要又标题，然后每个标题前面用“###标题” +换行。
示例如下:
剧本原稿：
```
“嘶……好冷！” 孟虎抱着胳膊，牙齿冻得“咯咯”作响，他呼出的白气像一条小龙，瞬间被江面上吹来的寒风撕得粉碎。
汤小团一个激灵，从昏沉中彻底清醒过来。眼前的一切让他把抱怨的话全吞回了肚子里。
他们正趴在一片冰冷潮湿的草地上，周围是黑压压的帐篷，一直延伸到远处昏暗的河岸边。数不清的火把在风中“噼啪”作响，将一个个手持长戈、面容冷峻的士兵的影子拉得又细又长，像张牙舞爪的怪物。空气中弥漫着一股刺鼻的味道，混合着泥土、汗水和某种金属的腥气。
“我的天，这……这是哪儿？” 唐菲菲扶了扶根本不存在的眼镜，小声惊呼。她一向镇定的小脸上，此刻也写满了震惊。
三人面面相觑，发现彼此身上都套着又脏又破的古代兵服，袖子长得能唱戏，裤腿短得像七分裤，看起来滑稽又狼狈。
“博物馆！那个破掉的陶锅！” 汤小团瞬间反应过来，压低声音道，“我们又穿越了！孟虎，你这乌鸦嘴，刚还在博物馆说想看看真的古代战场长什么样！”
“我哪知道摸一下锅的碎片，就能买到‘沉浸式体验’的VIP站票啊！” 孟虎哭丧着脸，揉着被硌得生疼的屁股，“这体验也太真实了吧？连个新手教程都没有！差评！”
唐菲菲迅速蹲下身，捻起一点地上的泥土，又警惕地观察着四周士兵的服装和旗帜，她的“学霸雷达”已经全面启动。
“注意看他们的发髻、盔甲样式，还有远处那杆迎风飘扬的大旗……” 她压着嗓子，语气却无比肯定，“如果我没猜错，我们来到了秦朝末年。这里是漳水河畔，巨鹿之战前夕，项羽的楚军大营！”
“项羽？就是那个‘力拔山兮气盖世’的西楚霸王？” 汤小团的眼睛瞬间亮了，好奇心压倒了恐惧，“哇塞！酷毙了！那我们这次的任务是什么？”
话音刚落，他们手腕上那块熟悉的学生手表微微一热，屏幕上浮现出一行歪歪扭扭的古篆字，下面跟着一行他们能看懂的白话文注解：
**【任务触发：见证历史——破釜沉舟】**
**任务说明：潜入楚军大营，找到项羽的帅帐，亲眼见证他下达“皆沉船，破釜甑”的命令。**
**回归线索：完成见证后，线索将自动浮现。**
“破釜沉舟？” 孟虎挠了挠头，“我知道这个成语，就是把做饭的锅都砸了，把渡河的船都弄沉，不留退路，跟敌人拼命！” 他顿了顿，脸色一变，“等等！砸锅？那我们岂不是没饭吃了？”
“吃货，现在是关心吃饭的时候吗？” 汤小团给了他一个“白眼”，“重点是，我们得在几万个古代士兵的眼皮子底下，找到项羽！这可比在游戏里找BOSS难多了！”
唐菲菲一脸严肃地补充道：“而且是‘潜入’，不是‘拜访’。我们现在这身打扮，一看就是‘炮灰’级别的小兵，乱闯军营，被当成逃兵或者奸细抓住，后果不堪设想。古代军法可是很严的，搞不好要被‘咔嚓’掉！” 她比划了一个砍头的手势。
```
